<br />

A rust trait layer, that defined the common behaviours of diversied types of heterogeneous hardwares. This module will unifies the interfaces btw. devices and build a conceptual interfaces upon them. The user of this module will see a fairly unified and compact abstraction of heterogeneous devices to built with.
